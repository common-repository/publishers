== Description ==

Companion plugin for the [Publishers theme](https://wordpress.org/themes/publishers/).

Includes: shortcodes, profile and post editor enhancements, membership control, and other features for improved editing and management of multi-author teams.

*Note: There are some cool features here that will work in other themes too, but they've been designed specially for the Publishers theme.*

**Features:**

* Add SEO, social media, and schema tags
* If you would like to enable the dark mode feature, simply add a new menu item to any of your menus (and give it the class *lights* and `#` for the URL)
* You can also add the dark/light mode switcher as a button anywhere you like and style it with `<span class="lights"><a href="#" class="button black">Lights</a></span>`
* Use the shortcode `[non-member]...[/non-member]` to show content to logged-out visitors only
* Use the shortcode `[member]...[/member]` to show content to logged-in members only
* Use the shortcode `[access capability="read"]...[/access]` to show content to users with [specific capabilities](https://wordpress.org/support/article/roles-and-capabilities/)
* Use the shortcode `[share]` to add share buttons to posts
* You'll find new optional profile fields: Public Email, Facebook, Twitter, Instagram, Pinterest, YouTube, and Twitch
* Add a body class for the logged-in user role
* Improved post editor tools/buttons
* Add subtitles/sub-headers to posts
* Feature posts on homepage
* Show a link to editor notes and guidelines in the post editor
* Uploaded images automatically have titles and alts generated
* Stop contributors from being able to delete files
* Receive an email for guest post submissions, and guests receive a follow-up email based on whether you published or deleted their submission
* Uncluttered admin and post editor screens for authors
* The /author/ slug in URLs is rewritten to /writer/
* New users are automatically logged in when they register
* Improved images handling: unnecessary files sizes aren't generated, thumbnails are generated for RSS feeds
* Users are sortable by registration date
* Records user IPs to assist with anti-spam measures and security whitelisting
* More to come...
